v1.14.0

Release 1.14.0


Editing Improvements

  • On the Edit Implementation Guide screen, a Changed! icon appears beside the implementation guide (IG) name when any edit is made before Save is clicked.
  • A pop-up warns you of unsaved content when you navigate from the Edit Implementation Guide screen to confirm you wish to discard changes.
  • In the Constraints Editor, re-slicing now follows the correct syntax and the content on the RAW tab matches the Element user interface.

Resources

  • In the Edit Implementation Guide's Resources tab, we added a new Sort button next to hand icon for adding resources. Click the Sort button to alphabetize resources.
  • Click the Sort button twice to reverse the sort.
  • On the Other Resources screen, we've added a History tab that functions like the History tab for Profiles, ImplementationGuide editing, etc.

Publishing

  • All required fields for publishing are now on the General tab in the Edit Implementation Guide screen.
  • The copyrightyear and releaselabel are inserted automatically during publishing, so you don't have to complete those fields.
