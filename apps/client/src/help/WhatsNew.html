What's New

Release 3.2.2

Released on 12/6/2023. Patched on 12/7/2023, 12/13/2023.

ValueSet Binding to VSAC For Dependent IGs

If an Implementation Guide (IG) is dependent on the Value Set Authority Center (VSAC), a user can bind an element to a ValueSet by selecting Constraint Manager>Binding>Value Set. A new VSAC tab is available that allows the user to search for an existing Value Set by name or by object identifier (whichever the VSAC API or package supports). Users are no longer required to use an external source to find the URL, copy it, and paste in into the ToF tool.

Support for Modifications to the Base Profile Slice

There are two updates to the Constraint Manager functionality:

  • When the user selects an element that has not been constrained, if the element already represents a slice, the user is presented with a new “Re-slice this Element” button
  • If the user selects the “Constrain this Element”, the tool now creates an element with a matchid/path when the base element is selected to allow the user to specify changes to the base element (i.e.. changing the minimum and maximum cardinalities).
