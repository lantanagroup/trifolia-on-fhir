v1.10.0

Release 1.10.0

Select a Published Implementation Guide (IG) as a Dependency


We have added a new icon to the dependency list. Users can select dependencies from existing IGs. 

  • Click the hand icon to display IGs from fhir-ig-list.json. 
  • Select an IG, and then select the IG version.

Display Successfully Published IGs

When opening a new file (project), users are now able to see the publishing status of each IG on the server based on the status of last publishing attempt. The status will be based on last IG publisher running, which IGs finished with a code of 0, and which IG finished with a code of not-zero.
