v1.12.0

Release 1.12.0


Editing Element Constraints

You can now select or edit these element constraints:

  • Key
  • Severity
  • Human
  • Expression


Each of these four elements constraints have an editable text box that allows users to input values into the @value.


Slice Name Warning

A warning now displays when you create a slice name containing spaces or periods.
